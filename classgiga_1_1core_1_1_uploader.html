std::vector<std::shared_ptr<FileUploader> > giga::core::Uploader::uploadingFiles ( )

Gets the uploading files.

During the uploading process, the local files are scanned. The scanned files are prepared (see isPreparationFinished()). When the preparation phase is finished a FileUploader object is created. This method return the list of FileUploader object already created.

In this list of FileUploader, at most one is currently being uploaded, some might be finished, other might be waiting.
